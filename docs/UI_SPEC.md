# UI layout theo trang

## Design system

- Visual: dark premium SaaS, glass panels, gradient violet/cyan.
- Grid: max-width 1280–1500px; sidebar 256px; topbar 64px.
- Radius: 12px controls, 20px cards, 28px media preview.
- Trạng thái: violet processing, green success, amber draft/warning, red failure.
- Mobile: sidebar chuyển thành bottom/nav drawer ở phase hoàn thiện.

## Home

1. Sticky header: logo, workflow, templates, pricing, CTA.
2. Hero hai cột: value proposition + mock wizard preview.
3. 4-step workflow.
4. Template theo ngành.
5. Pricing cards.
6. FAQ và footer.

## Pricing

- Hero ngắn.

## Create Project

- Header: project name, autosave, estimated credit.
- Top progress bar + horizontal stepper.
- Left 55%: current-step controls.
- Right 45%: sticky live preview, model/output/credit stats.
- Bottom: Back/Continue; disable duplicate submit during generation.

### Step 1 Upload Product
- Dropzone, file constraints, product name, industry, quality tips.

### Step 2 Character Selection
- Character cards 2 columns; preview voice; custom character CTA.

### Step 3 Scene Selection
- Scene visual cards 2–3 columns; category badge.

### Step 4 Composite
- Summary, model, variants, generated prompt preview, processing state.

### Step 5 Video
- Duration, aspect ratio, motion script, video style.
- 15–30s surfaced as multi-shot (P1 orchestration).

### Step 6 Voice & Subtitle
- Voice dropdown, subtitle style, voiceover text, caption, hashtags.

### Step 7 Export
- Success summary, MP4/SRT buttons, caption and credit breakdown.
